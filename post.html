Implementing a CSS Prepocessor

Rename all css files to start with an _ and to the extension .scss. I like to put my .scss files in a scss folder.

_large.scss
_medium.scss
_normalize.scss
_small.scss

Create a _variables.scss file

Create a styles.scss file In this file add the @imports

@import "normalize";
@import "variables";
@import "small";
@import "large";

For Visual Studio Code search and install the Extension "Live Sass Compiler"
